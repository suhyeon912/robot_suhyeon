[네이버 뉴스 요약 페이지 요구사항]

1. 각 분야별 최신 뉴스를 하나 선정하여 각 페이지 구성
- 페이지 구성 :
    1) 메인 페이지
    2) 정치 페이지
    3) 경제 페이지
    4) 사회 페이지
    5) IT/과학 페이지
    6) TV/연예 페이지

2. 메인 페이지는 분류별 링크를 포함
    상단과 하단 디자인은 각 페이지에서 유지

3. 뉴스 하단에 관련 뉴스를 링크

4. 전체 공통 메뉴는 각 페이지로 링크
